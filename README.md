# KASHILAV — Landing Page

Landing page da agência KASHILAV — marketing especializado para clínicas estéticas e turismo médico em Bogotá.

## Stack
Site estático (HTML + CSS + JS puro). Sem build. Publica direto na Vercel.

## Estrutura
- `index.html` — conteúdo da página
- `styles.css` — visual
- `script.js` — troca de idioma ES/EN

## Rodar localmente
Abrir o `index.html` no navegador, ou:
```
npx serve .
```

## Publicar
Conectado ao GitHub → Vercel publica automaticamente a cada push.

## A personalizar
- Número do WhatsApp em `index.html` (procurar por `wa.me/000000000000`)
- E-mail de contato (`contacto@kashilav.com`)
